# 🧠 Deep Agents Chatbot — Streamlit App

A conversational chatbot built on the `deepagents` library that demonstrates **all major features** from the demo notebooks:

- **Basics** → custom models, system prompts, tools (Tavily web search), planning (`write_todos`), virtual files  
- **Context engineering** → AGENTS.md context, memory, checkpointer, thread conversation memory, skills  
- **Backends** → StateBackend, FilesystemBackend, StoreBackend  
- **Subagents** → research-agent, structured output with Pydantic schema  

---

## 🚀 Features
- **[Custom tools](ca://s?q=Deep_agents_custom_tools)**: Internet search via Tavily API  
- **[Context engineering](ca://s?q=Deep_agents_context_engineering)**: AGENTS.md + skills seeding  
- **[Multiple backends](ca://s?q=Deep_agents_backends)**: State, Filesystem, Store  
- **[Subagents](ca://s?q=Deep_agents_subagents)**: Research agent + structured researcher with confidence scores  
- **[Thread memory](ca://s?q=Deep_agents_thread_memory)**: Persistent conversation state with checkpointer  
- **[Virtual file system](ca://s?q=Deep_agents_virtual_file_system)**: Store and manage files in agent state
